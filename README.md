# NOVELNET
This Project used these following things:-</br>
1> React.js</br>
2> tailwind css.</br>
3> express,js </br>
4> mongodb </br>
5> imagekit </br>
6> install "Atom" for recoil</br>

other things but important one :-</br>
- For Validation -> ZOD used.</br>
- For auth -> JWT. </br>
- For file/form -> Multer </br>
- For API testing -> Postman </br>

# what u need to do :-</br>
1> Create Account on the ImageKit.io</br>
2> Take importent keys that need to run the project from imagekit and put it in the .env file of your project backend.</br>
3> In .env file you need these following thing:</br>
- JWT_Seceret</br>
- imagekit keys</br>
- mongodb key</br>
4> Install the dependencies by running -> npm i or npm install </br>
5> TO RUN THE PROJECT FRONTEND -> npm run dev; and FOR BACKEND -> node index.js </br>
